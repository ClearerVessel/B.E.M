# BEM-pswd-Vault
2fA password repository, first try making a program

Allows you to compile and encode passwords into a single file locally on your computer. creates a unique key code to be kept on a drive of your choosing. It currently saves the the file to whatever location the application is being run on. Next iteration will let you choose location and name of document on local drive.

FOR PC use:
you should be able to just download and run the application


if that does not work OR use with mac follow these steps

1. Ensure python loaded to computer
2. open terminal in folder
3. Install the encryption library

  	pip install -r requirements.txt

4. Run the vault program

    double click BEM.bat file for your system
	
-or in terminal type
	
    py BEMv2.py
